Android Разработка

Android позволяет создавать Java-приложения, управляющие устройством через разработанные Google библиотеки. Android Native Development Kit позволяет портировать библиотеки и компоненты приложений, написанные на Си и других языках. В Android используется специальная виртуальная машина под названием Dalvik Virtual Machine. Дальвик использует свой собственный специальный байт-код. Вот почему вы не можете использовать стандартный байт-код Java на Android. Android предоставляет инструмент "dx", который преобразует файлы классов Java в файлы "dex" (Dalvik Executable). Приложения для Android .в APK-файлах (Android Package), упакованных "aapt" (Android Asset Packaging Tool), Google предоставляет инструменты разработки Android (ADT) для Eclipse для упрощения разработки. ADT автоматически преобразует файлы классов Java в файлы dex и создает apk во время развертывания. Каждое приложение Android работает в своем собственном процессе и под своим собственным идентификатором пользователя, который автоматически создается Android во время развертывания. Поэтому приложение будет отключено от других запущенных приложений, а неработающее приложение не сможет беспрепятственно повредить другие приложения Android. Любое приложение для системы Android можно легко создать, установив свободно доступный SDK. Эти программы поддерживают любое устройство, установленное на операционной системе Android и написанное на Java. Они работают на виртуальных машинах Java (JVM).
